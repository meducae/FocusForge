**FocusForge** is an innovative productivity ecosystem built to help users break free from social media addiction. Unlike traditional blockers, it enforces a **Dopamine Detox** by requiring physical or intellectual "work" to earn digital currency, which can then be used to "buy" time on distracting apps.

## ✨ Core Features

*   **🚫 Smart App Blocker**: Deep system integration to monitor and restrict access to addictive applications (Social Media, Games).
*   **🧘‍♂️ AI-Powered Exercise Validation**: Real-time pose estimation using ML to verify that you're actually doing your squats or push-ups before getting your reward.
*   **🎙️ Speech-to-Earn**: Integrated English speech evaluation system. Practice your language skills to unlock your favorite apps.
*   **💎 Gamified Economy**: Earn "Focus Coins" through healthy habits. Spend them in the **Reward Shop** to temporarily bypass blocks.
*   **🎨 Premium Glassmorphism UI**: A stunning, Apple Vision Pro-inspired interface with translucent panels, neon glows, and fluid animations.
*   **🏗️ Clean Architecture**: Built with a modular approach (Domain, Data, Presentation) ensuring scalability and high code quality.

## 🛠 Tech Stack

- **Mobile**: Kotlin Multiplatform (KMP), Jetpack Compose, Koin (DI), Coroutines & Flow.
- **Backend**: Python (FastAPI), WebSockets for real-time AI processing.
- **AI/ML**: Custom Pose Tracking & Speech Recognition models.

## 🚀 Impact
This project isn't just an app; it's a statement against the attention economy. It turns the phone from a distraction device into a tool for self-improvement.
